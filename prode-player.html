📖 How to Play

Predict the score for every match before the admin locks the round.

🥇Exact score → 5 pts
🥈Right result + goal diff → 3 pts
🥉Right result only → 1 pt
❌Wrong result → 0 pts
⚽ Bracket Matches
Score is counted at the end of 90 or 120 minutes — before penalties. A match decided on penalties is scored at the end of extra time (e.g. 1-1 counts, not the penalty shootout).
TIE-BREAKER ORDER
1️⃣Most correct results (any tier)
2️⃣Most exact scores
3️⃣Most correct goal differences
=Perfect tie shown with = badge
